React app oluşturmak için(Windows)node.js'i indirmemiz gerekiyor. (İndirmek için:https://nodejs.org/en/download/) indirdikten sonra cmd'yi açıyoruz. Ve bu kodları sırası ile çalıştırıyoruz. 
npx install -g create-react-app
npx create-react-app my-app
cd my-app
npm start
React App oluşmuş olucak.Sitede değişiklik yapmak istiyorsanız "src" klasörü içinde App.js dosyasını editörünüz ile açın.

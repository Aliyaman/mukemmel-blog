Uzun Bir Süreden Sonra Tekrar Merhabalar Bugün Tor Ağı Üzerinde Nasıl .onion uzantılı Websitesi Açarsınız ona bakacağız.

Hadi Başlayalım

Bunun ilk yolu tor ağı üzerinde bulunan diğer websitelerinden hosting hizmeti almak bunun ücretlileri ve ücretsiz yolları var ben ücretsiz yoldan ve kendi bilgisayarınız üzerinden bunu yapmadan bahsedeceğim.

Daniels Hosting

Ilk yol Daniels Hosting buradan kendi Website'nizi ücretsiz bir şekilde yayınlayabilirsiniz. Bugün Bunu anlatmayacağım ona buradan bakabilirsiniz. Türkçe Değil ama ordaki adımları takip ederseniz yapabilirsiniz.

Ikinci Yol Kendi Makineniz

Adım 1

Toru indirmek!

~$ sudo apt install tor
[sudo] password for kali:

Adım 2

Nginx'i indirmek

Nginx bir http web serverdir bunu sayesinde .html dosyamızı domainimizde gösterebileceğiz.

~$ sudo apt-get install nginx
[sudo] password for kali:

Adım iki

torrc dosyasını güncellemek.

~$ cd /etc/tor/
~$ nano torrc


Aşağı dogru indiğinizde böyle bir kısım göreceksiniz.

############### This section is just for location-hidden services ###
## Once you have configured a hidden service, you can look at the
## contents of the file ".../hidden_service/hostname" for the address
## to tell people.
##
## HiddenServicePort x y:z says to redirect requests on port x to the
## address y:z.

#HiddenServiceDir /var/lib/tor/hidden_service/
#HiddenServicePort 80 127.0.0.1:80

#HiddenServiceDir /var/lib/tor/other_hidden_service/
#HiddenServicePort 80 127.0.0.1:80
#HiddenServicePort 22 127.0.0.1:22

Adım 3

Bu iki satırın başındaki # sembolü kaldırıp açıklama satırı halinden çıkaracağız.

#HiddenServiceDir /var/lib/tor/hidden_service/
#HiddenServicePort 80 127.0.0.1:80

Adım 4

Ardından tor servisini başlatacağız.

~$ sudo service tor start
[sudo] password for kali:

Şimdi .onion linkimizi öğreneceğiz.

~$ cd /var/lib/tor/hidden_service/

Burada iki dosya göreceğiz hostname dosyası bizim domain adresimiz.

~$/var/lib/tor/hidden_service# ls

hostname private_key

Adım 5

hostname dosyasına bakalım.

~$/var/lib/tor/hidden_service# cat hostname

SizinDomainAdresiniz.onion

Adım 6

Web serverimizi başlatalım.

~$ service nginx start

Bukadar artık tor browseri bilgisayarınıza kurup websitenize bakın websitenizin html dosyası "/var/www/html/" klasörü içindedir. Yoksa kendiniz olusturun.
